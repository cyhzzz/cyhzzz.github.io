01关于我

我是一名产品经理，近两年做的事就两件——把互联网渠道拓客做透、把 AIGC 用到一线。

渠道侧：从 0 到 1 搭建公司级互联网渠道投放获客体系，覆盖巨量引擎、小红书聚光、腾讯广告及主流应用市场，目前月耗百万级。搭建数字化采集监测分析体系，打通「开户 → 入金 → 交易 → 资产沉淀」全链路转化，以数据驱动策略快速调优。兼具自主操盘与供应商管理经验。

AI 侧：主导财富管理领域投顾内容的 AIGC 全链路赋能——涵盖评论文章、热点解读、图片笔记等多渠道多模态内容，从选题、辅助创作到 AI 合规预审贯通全流程。同时面向总部运营团队、外包设计团队、分支投顾账号开展 AIGC 培训，把 AI 能力沉淀为可复用 Skill。也兼顾一线：曾挂职分公司财富管理部一年，跑过营业部、见过真实客户。

github.com/cyhzzz/shengxintou-bi · JavaScript / Python
★ 与算法投放/智能投放 JD 高度契合

轻量级互联网广告投放分析平台，是互联网渠道拓客全链路数据支撑体系的落地 Demo。整合腾讯广告、抖音广告、小红书广告三大平台数据，提供多维度投放分析与可视化，覆盖「消耗 → 转化 → 投产」全链路监测。

通过这个项目，探索各投放平台与内部数据结构、数据链路，并尝试用 vibe coding 辅助需求推进与运营分析——把 AI 写代码的能力用到产品迭代和数据分析的一线。
